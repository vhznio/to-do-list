## Getting Started

First, install dependencies:

```bash
npm install
```
Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
Third, create and populate a database.

```bash
npx prisma generate
npx prisma migrate dev
npx prisma db seed
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
